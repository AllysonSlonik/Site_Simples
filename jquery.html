Jquery Animate Function

To do it you have to use the document.ready in order to load the elements of the page first and then start using them. Inside you use the animate() function. $(document).ready(function(){ $("#btnDown").click(function(){ $("#square").animate({top: '250px'}); }); });

Focus, Blur and Hover

They don't need the animate() function to work, but you can use it if you want to animate something.

Focus and Blur are for forms, you can see more on the focus() section.

Hover, whenever you hover over an item, it will perform some action, the coding for it is similar to blur and focus since you DON'T need to use the animate() function.

focus() functon = whenever you click inside a form field it can change its state. blur() whenever you click outside the form field it change its state.
